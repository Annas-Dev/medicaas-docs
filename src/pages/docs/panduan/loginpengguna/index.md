## Login Pengguna

1. Buka browser, Kemudian akses **halaman medicaas anda** (http://**nama domain**)
2. Masukkan **username** pengguna
3. Masukkan **password** pengguna
4. Clik tombol masuk dan tunggu sampai di alihkan ke halaman dashboard
